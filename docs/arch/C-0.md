# C-0 — CoAgent Family Scaffold + Core

Date: 2026-04-26

## What landed

`electron/native/coagent/` — types + bus + core cell:

- **`types.ts`** — `COAGENT_FAMILY_ID = 'wish.coagent'`, `CoAgentRole`
  (`core` / `task` / `deliverable` / `approval` / `knowledge` /
  `activity` / `ui` / `orchestration`), `CoAgentEventKind` (14
  stable kinds), `CoAgentEvent` shape.
- **`bus.ts`** — `CoAgentBus` is a typed pub/sub: members declare
  subscribed kinds at join, the bus filters before delivery, and
  thrown handlers are caught + logged via Tel-0
  (`coagent.subscriber.threw`). `CoAgentCore` joins the bus as `core`
  and keeps a 256-event ring for diagnostics + UI replays.

## Bus contract

| Operation | Effect |
|---|---|
| `bus.join(member, handler)` | adds/replaces a member; first join emits `family.member.joined` |
| `joinResult.publish(ev)` | publishes with `source = role`; `ts` auto-stamped |
| `joinResult.leave()` | emits `family.member.left` |
| `bus.publish(role, ev)` | host-level publish (used by tests + adapters) |
| `bus.subscribersFor(kind)` | diagnostic; lists roles subscribed to a kind |
| `bus.membership()` | snapshot; deep-copied |

Last-writer-wins on re-join (matches Cell-2 deactivate→activate).
Re-joining the same role does NOT re-emit `family.member.joined`
(noise control).

## Telemetry

Every publish emits `coagent.event.delivered` (`debug`) carrying
`{ kind, source, recipients }`. Thrown subscribers emit
`coagent.subscriber.threw` (`warn`) carrying `{ role, kind, reason }`.
