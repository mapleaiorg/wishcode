# K-2 — Knowledge Retrieval

Date: 2026-04-27

## What landed

`electron/native/knowledge/retrieval.ts` (~90 LoC) — `Retriever` joins
K-1 BM25 ranking with K-0 source filters (scope + status + slug
allow/deny + source-id allow). Used by C-4 (team knowledge) and the
agent runtime (A-2).

## API

```typescript
new Retriever(store, indexer)
retriever.retrieve({
  query, limit?, minScore?,
  scope?,                     // narrow eligible sources by scope
  status?,                    // default ['indexed']
  slugAllowList?, slugDenyList?,
  sourceIdAllowList?,
}) → { hits, stats }
```

`stats` reports `totalIndexedSources`, `eligibleSources`, `indexHits`,
`droppedByScore`, `droppedByPolicy` for the future `/retrieval`
sidebar.

Every hit carries the K-1 `ProvenanceRef` shape — ready to pass to
`Cell-3.knowledge.cite()` without translation.
