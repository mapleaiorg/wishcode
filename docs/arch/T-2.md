# T-2 — Activity Timeline

Date: 2026-04-26

## What landed

`electron/native/activity/timeline.ts` (~110 LoC) — `ActivityTimeline`
implements `TelemetrySink` and groups events by `traceId` into
UI-friendly `TimelineEntry` records. Tracks per-trace `highestLevel`,
distinct domains (lowercase first segment of event type), and
newest-first ordering with insertion-seq tiebreak.

Caps: `maxEventsPerTrace` (default 200, FIFO drop), `maxTraces`
(default 100, oldest-evict). `clear()` for tests; `trace(id)` and
`list({ domain?, minLevel? })` return deep copies.

Every existing producer feeds in for free — T-1 (`job.run.*`),
Cell-2 (`cell.activate.*`), D-6 (`capability.*`), Tel-0 (any caller).
